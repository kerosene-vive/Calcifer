# Calcifer Nano

## Setup

```bash
npm i
npm run build
```

Load the `dist` folder as unpacked extension in Chrome.


---
## Requirements

Developed and tested with Google Chrome
Version 131.0.6778.33 (Official Build) beta (x86_64)
